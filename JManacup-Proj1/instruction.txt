HOW TO RUN THE PROGRAM
John Manacup
03/13/2021

Requirement:

Have a c++ compiler preferably g++.

The program is done through C++.

To run the program,

Type in the terminal

        g++ proj1.cpp -o proj1

Then,

        ./proj1 <input file name>


EXAMPLE

If the file is "foo.txt" then it will be,

        g++ proj1.cpp -o proj1
        ./proj1 foo.txt
